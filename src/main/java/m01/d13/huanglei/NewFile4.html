         叶新童，出生于1963年3月8日，中国香港男演员[1] 。 1982年，叶童在自己的第一部影片《烈火青春》担任女主角并获得金像奖最佳新人提名[2] ；1983年因主演喜剧电影《表错七日情》 而获得香港电影金像奖最佳女主角[3] ；1990年凭借电影《飞越黄昏》获得金像奖最佳女配角[4] ；1992年出演电视剧《新白娘子传奇》中的许仙[5] ；1994年主演电视剧《倚天屠龙记》[6-7] ；1997年，与刘青云共同主演喜剧电影《对不起，多谢你》[8] ；2012年，出演根据游戏改编的电视剧《轩辕剑之天之痕》[9] ；2014年，叶童参演由段奕宏、戚薇主演的《谋圣鬼谷子》[10-11] 。2016年，参演由阮经天、杜鹃主演的爱情电影《纽约纽约》[12] 。
